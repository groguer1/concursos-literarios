Responsable del tratamiento

         

David Mateos Pascual, email: dmateos.pascual@gmail.com, es el responsable del tratamiento de los datos personales en letrasespanolas.org.

         

Datos que recopilamos

         

Este sitio web no recopila datos personales de forma directa. No existe formulario de registro ni suscripción. Se procesan automáticamente:

         
  • Datos de navegación: Dirección IP, navegador y páginas visitadas, de forma anónima mediante GitHub Pages.
  • Almacenamiento local: El navegador guarda en localStorage la caché de concursos durante 24 horas. Estos datos no se transfieren a ningún servidor externo.
